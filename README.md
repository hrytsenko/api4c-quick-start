# API4C Bootstrap

## Flow

1. Choose format for contracts.
1. Share contracts via *repo*.
1. Enforce rules via *lint*.
1. Detect changes via *diff*.
1. Validate contracts via *proxy*.

## Tools

1. [OpenAPI](https://www.openapis.org/) as API specification.
1. [Apicurio](https://www.apicur.io/registry/) as API registry.
1. [Spectral](https://stoplight.io/open-source/spectral) as rule linter.
1. [Oasdiff](https://www.oasdiff.com/) as change detector.
1. [Prism](https://stoplight.io/open-source/prism) as validation proxy.
1. [Mockoon](https://mockoon.com/) as mock server.
